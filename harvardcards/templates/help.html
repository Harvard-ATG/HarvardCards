HarvardCards

HarvardCards is a flashcard application to help learners memorize vocabulary, images, or other information.

The Basic Concepts

HarvardCards has three key concepts that are helpful to understand: collections, decks, and cards.

Collection: The first step an instructor will take when setting up HarvardCards will be to create and name a collection. A collection is like the "folder" concept in personal computers; however, instead of containing files, a Collection will contain decks. A collection will often correspond to a particular course (and will often be given the name of the course, such as "French 101" in the example above). Only after the collection is created can the instructor create decks of cards that will exist within that collection.

Deck: As previously mentioned, a collection may contain any number of decks, which are groups of cards. In the French 101 example, a deck was created that contains all the verbs from Chapter 1, and was given the name, "verbs". A second deck, "nouns", was also created.

Card: Just as collections contain decks, decks contain cards. A card is the most basic item in the application, and is analogous to a 3 x 5 card you may have used as a student. In the above example, the deck, "verbs," contains many cards, one for each verb to be studied.

To recap: A Collection is a grouping of any number of decks, and often corresponds to a course. A deck is a group that can contain any number of cards; usually a deck contains a logical grouping, such as vocabulary from a particular chapter. A card, the most basic unit, corresponds to an item being studied, such as a vocabulary word, or an image in an art history class. When using the application, the steps to set up your course will be to first create a collection, then create one or more decks. Finally, you will either manually add cards to the deck, or batch upload cards to a deck.

Getting Started

  1. Setting up a collection: When you log in as an administrator, the first thing you will need to do is set up a collection to contain your deck(s). Here are the steps required:
    1. Click on "Add Collection."
    2. Enter the name of the collection in the text box that says "Enter course name here". (Typically, a collection will correspond to the name of a course.)
    3. Select a Template from the dropdown list. Since this application will be used for various types of classes, we have created a few templates for you to choose from. The template will determine the layout of the card (both front and back), as well as what type of information displayed. For example, if you select the "Art History" template, there will be a place for you to display an image. If you select one of the foreign language templates, there will be a place for pronunciation.
    4. Click Submit.
  2. Adding a deck: Once you have created a collection you can add any number of decks. To create a deck:
    1. Click on "Add a Deck."
    2. Click on the text that says, "Untitled Deck," and type in a name.
  3. Adding cards: Once you have created a deck, you can add cards to it. There are two ways to add card: You can manually enter information, or you can upload many cards at once via an Excel spreadsheet.
    1. Adding cards manually:
      1. Click on "Add Card."
      2. Fill out fields that appear (such as "Term"), on the front of the card, then fill out the fields (such as "Definition") on the back of the card.
      3. If you want, select a card color.
      4. Click Submit.
    2. Uploading cards:
      1. Click on "Upload Cards."
      2. Download a copy of the "flashcard template." Fill out the rows with the data you want to import. Save your file.
      3. Click on "Browse..."
      4. Select the file to upload and hit Submit.
      5. You should see a message confirming success.

A word about templates

When you set up a Collection, you are prompted to choose a Template. A template determines the fields that will display on the cards in the decks in that collection. In the Art History Template, for example, there is a place in the layout of the card that will display an image.

The current templates are as follows:

  1. Default: A basic template that allows a term on the front of the card, and a definition on the back. Useful for learning vocabulary.
  2. Art History: A Template for studying images. An image can be displayed on the front. The fields on the back can include Artist, Date, Title, Materials and Location.
  3. Language with Audio: A Template for studying terms with accompanying audio. It could be used to study vocabulary along with pronunciation. The front can display a term, and a sound recording. The back displays the definition.
