HarvardCards

HarvardCards is a flashcard application to help learners memorize vocabulary, images, or other information.

The Basic Concepts (Collection/Deck/Card)

HarvardCards has three key concepts that are helpful to understand, Collections, Decks, and Cards.

Collection: The first step an instructor will take when setting up Harvard Cards will be to create and name a Collection. A Collection is like the "folder" concept in personal computers. However, instead of containing files, it will contain "Decks". A Collection will often be created that corresponds to a particular course (and will often given the name of the course, such as, French 101). Only after the Collection is created, will the instructor create Decks of Cards that will exist within that Collection.

Deck: As mentioned above, a Collection will contain any number of Decks. After the instructor creates a Collection, in the example, let's say the Collection is named, "French 101". The next step will be to add one or more Decks, which correspond to groups of cards. So to extend the French 10 example, a deck may be created the contains all the verbs from Chapter 1, and that Deck might be given the name, "verbs".

Card: Just as Collections contain Decks, Decks contain Cards. The Card is the most basic item in the application, and is analogous to a 3 x 5 card you may have used as a student. So, in the above example, where a Deck has been created called "verbs", you can imagine individual Cards being created, one for each verb to be studied.

To recap: A Collection is a grouping of any number of Decks, and often corresponds to a course. A Deck is a grouping containing any number of Cards, usually a deck contains a logical grouping, such as vocabulary from a particular chapter. A Card, the most basic unit, corresponds to an item being studied, such as a vocabulary word, or an image in an art history class. When using the application, the steps to set up your course will be to create a Collection, then create one or more Decks. Finally, you will either manually add cards to the Deck, or batch upload cards to a Deck.

A word about templates

When you set up a Collection, you are prompted to choose a Template. A template determines the fields that will display on the Cards in the Decks in that collection. In the Art History Template, for example, there is a place in the layout of the card that will display an image.

The current templates are as follows:

  1. Default: A basic template that allows a term on the front of the card, and a definition on the back. Useful for learning vocabulary.
  2. Art History: A Template for studying images. An image can be displayed on the front. The fields on the back can include: Artist, Date, Title, Materials and Location.
  3. Language with Audio: A Template for studying terms with accompanying audio. It could be used to study vocabulary along with pronunciation. The front can display a term, and a sound recording. The back displays the definition.
